# Copywriting: Voice & Tone Guidelines

When generating text for ARyCP, adhere to this brand persona.

## Language
* **Primary language:** Spanish (México). All user-facing text must be in Spanish.
* **Technical terms:** Use accepted Spanish equivalents when available. Keep widely understood English terms (e.g., "dashboard", "login") only if the Spanish alternative is awkward.

## Brand Personality Keywords
* Professional but approachable
* Direct and efficient
* Knowledgeable in accounting, fiscal, legal, and labor topics
* Empathetic — users are busy professionals managing complex processes

## Grammar & Mechanics Rules
* **Headings:** Use Title Case for main headings (H1, H2). Use sentence case for subheadings (H3+).
* **Punctuation:** Avoid exclamation points (!) in standard interface copy. Use periods for complete sentences.
* **Clarity:** Prefer active voice over passive voice. Keep sentences concise.
* **Formality:** Use "tú" (informal) for UI interactions. Use "usted" only in formal legal or client-facing documents.

## Terminology Guide

| No Usar | Usar en su Lugar |
| :--- | :--- |
| "Utilizar" | "Usar" |
| "Con el fin de..." | "Para..." |
| "Eliminar" (para acciones reversibles) | "Quitar" |
| "Error fatal" | "Algo salió mal" |
| "Ejecutar" (para acciones de usuario) | "Realizar" o "Hacer" |
| "Trámite" (genérico) | Nombre específico del trámite (e.g., "Alta patronal IMSS") |
